# ai-narrative — конвейер ИИ-ботов для визуальных новелл 18+

Пайплайн на **LangGraph** генерирует визуальную новеллу (русский, жанр adult)
от идеи до готового текста: логлайн → синопсис → персонажи → структура →
поглавное написание (диалоги + адалт) → редактура → перевод. Управление и
ручные правки на каждом шаге — через веб-интерфейс (**Next.js + FastAPI**).

Каждый бот передаёт результат следующему через **общую память** (LangGraph
state + SQLite-чекпоинт). Редактор проверяет каждую главу и **гонит правку
обратно** до полного устранения замечаний. Откровенные сцены пишет отдельная
модель без цензуры (структурные боты могут работать через **подписку Claude**
без оплаты токенов).

---

## Возможности

- **Поглавный конвейер с гейтом качества**: глава пишется → сразу проверяется
  редактором → правки до закрытия всех замечаний → только потом следующая.
- **Адалт в каждой главе**: откровенная сцена встроена в диалоги (модель без
  цензуры), с канон-гардом характеров и контролем объёма.
- **Точечная правка**: редактор-замечания исправляются адресно, без
  перегенерации главы; цикл «исправить и перепроверить» с чатом по правкам.
- **Веб-UI**: пошаговый режим, ручное редактирование артефактов, выбор
  логлайна, апрув числа глав, чат с ИИ по главе, откат этапов, live-прогресс.
- **Экспорт готового проекта** в `.txt` / `.md`.
- **Claude по подписке** (Pro/Max) для ботов 1–5 и редактора — без оплаты API.
- **Устойчивость**: долгие LLM-операции в фоне (нет таймаутов прокси),
  обработка ошибок провайдера, восстановление прогона из чекпоинта после краша.

## Конвейер

```
Логлайн → Синопсис → Персонажи → Объём(апрув) → Структура → Ред.структуры
   │
   └─ поглавный цикл (одна глава за раз):
        Диалоги+Адалт → Редактор ──┐
            ▲                       │ есть открытые замечания
            └── точечная правка ◄───┘ (исправить/отклонить → перепроверить)
        │  глава доведена (0 открытых) → следующая глава
   │
   └─ Перевод (опц.) → ✅ Проект готов → скачать .txt/.md
```

| Шаг | Узел | Модель |
|---|---|---|
| Логлайн / Синопсис / Персонажи | `logline` `synopsis` `characters` | Claude (OpenRouter или подписка) |
| Объём (ИИ предлагает число глав) | `chapter_count` | Claude |
| Структура + редактор структуры | `structure` `structure_editor` | Claude |
| Диалоги + адалт (внутри главы) | `dialogue` | структурная часть — Claude; адалт — **grok (без цензуры)** |
| Редактор главы | `editor` | Claude (strict JSON-замечания) |
| Перевод | `translation` | Claude |

## Быстрый старт — Docker (рекомендуется)

Нужен Docker + Docker Compose.

```bash
git clone https://github.com/soca1m/ai-narrative.git
cd ai-narrative
cp .env.example .env          # впиши OPENROUTER_API_KEY (минимум)
docker compose up --build
```

Открой **http://localhost:3030**.

- Фронт публикуется на `:3030`, бэкенд наружу **не** выставляется (доступ только
  через прокси фронта — same-origin, без CORS).
- Стейт прогонов хранится в томе `narrative_data` (переживает пересоздание).
- **Подписка Claude в Docker**: compose монтирует `~/.claude` (read-only) — если
  ты залогинен в Claude Code на хосте, подписка заработает. Не нужна — закомmenти
  эту строку в `docker-compose.yml` (боты пойдут через OpenRouter).

## Локальная установка (без Docker)

Нужен Python 3.11+ и Node.js 20+.

**1. Бэкенд:**
```bash
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env          # впиши ключи
PYTHONPATH=src uvicorn server.app:app --host 127.0.0.1 --port 8000
```

**2. Фронтенд** (в другом терминале):
```bash
cd web
npm install
npm run dev -- -p 3030        # проксирует /api на 127.0.0.1:8000
```

Открой **http://localhost:3030**.

> CLI без веба (один прогон в файл): `python scripts/run.py --help`.

## Переменные окружения

Полный список с комментариями — в [`.env.example`](.env.example). Ключевое:

| Переменная | Назначение |
|---|---|
| `OPENROUTER_API_KEY` | **Обязательна.** Ключ OpenRouter (адалт + боты без подписки). |
| `CLAUDE_MODEL` / `EDITOR_MODEL` | Модели структурных ботов и редактора. |
| `ADULT_MODEL` | Модель без цензуры для адалта (дефолт `x-ai/grok-4.3`). |
| `USE_CLAUDE_SUBSCRIPTION` | `1` — боты 1–5 + редактор через подписку Claude. |
| `CLAUDE_SUB_MODEL` | `haiku` \| `sonnet` \| `opus` для подписки. |
| `MAX_REVISIONS` | Лимит авто-ревизий главы (в UI цикл без лимита). |
| `CORS_ORIGINS` | Разрешённые origin фронта (CSRF-защита бэка). |
| `NARRATIVE_DB` | Путь к SQLite-чекпоинту (в Docker → том). |
| `BACKEND_URL` | Куда Next проксирует `/api` (Docker: `http://backend:8000`). |

## Claude по подписке (без оплаты токенов)

Боты 1–5 и редактор могут работать через твою подписку **Claude Pro/Max**
(логин Claude Code), а не платный API. Адалт всё равно идёт через OpenRouter
(Claude его цензурит).

Включение: в UI панель **«Claude по подписке»** → тумблер «Все боты через
подписку» + выбор модели. Авторизация (если не залогинен):
1. Кнопка **«🔗 Авторизоваться»** → откроется/появится ссылка на страницу Claude.
2. Войди, подтверди доступ, скопируй выданный код.
3. Вставь код в поле → «Подключить».

Альтернатива — `claude setup-token` в терминале и вставить токен, либо просто
быть залогиненным в Claude Code (`~/.claude/.credentials.json`).

> `ANTHROPIC_API_KEY` перебивает подписку платным API — не задавай его, если
> хочешь именно подписку.

## Стоимость и выбор моделей

Связка по умолчанию (боты+редактор через подписку, адалт grok) — **~$0.06–0.16
за новеллу из 3 глав** (только адалт-токены OpenRouter). Чистый OpenRouter без
подписки — дороже. Сравнение моделей адалта (grok-4.3 лидер по RU+explicit,
grok-4.20 цензурит, deepseek медленнее, RP-тюны слабый русский) — в `docs/`.

## Структура проекта

```
src/narrative/      # пайплайн: graph, nodes (боты), prompts, state, llm, routing
  ├─ graph.py       # сборка LangGraph (поглавный цикл + правки)
  ├─ nodes.py       # боты: write_chapter, patch_chapter, editor_node, ...
  ├─ prompts.py     # системные промпты ботов + канон-гард адалта
  ├─ claude_sub.py  # LLM-обёртка над подпиской Claude (Agent SDK)
  └─ state.py       # State + Pydantic-схемы артефактов
server/app.py       # FastAPI: запуск/управление прогоном, правки, экспорт, OAuth
web/                # Next.js UI (app/page.tsx, lib/api.ts, app/globals.css)
scripts/            # e2e_test, serve_mock, real_run, test_adult_model, run
Dockerfile          # образ бэкенда
web/Dockerfile      # образ фронтенда
docker-compose.yml  # весь стек
```

## Безопасность

- `.env`, `*.db`, артефакты тестов — в `.gitignore`, в образ не попадают.
- Бэкенд слушает только локально (вне Docker) / внутри сети (в Docker), наружу
  не публикуется. CORS ограничен origin'ами фронта.
- Подписка Claude = индивидуальное использование (ToS Anthropic).
